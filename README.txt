INSTALLING
----------------------------------------------------------
1. You must have installed node.js, then updated npm 
https://docs.npmjs.com/getting-started/installing-node
2. In directory with files you open consol and install npm
(	1. then write "npm install";
	2. after installing node_modules write "npm start"; )
3. Then your default browser open local server with site.
